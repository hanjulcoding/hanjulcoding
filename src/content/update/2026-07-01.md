## Tools 메뉴

자주 쓰는 유틸을 모아둘 `Tools` 메뉴를 추가했다. posts처럼 목록(검색·태그·리스트)이 있고, tool 하나가 페이지 하나로 붙는 구조다. 각 도구는 React 아일랜드로 동작하고 데이터는 브라우저(localStorage)에 저장된다.

### MD to Text

Gemini·ChatGPT가 뱉는 마크다운 문법을 걷어내는 도구.

- 굵은 글씨·헤더·리스트 등 마크다운 서식 제거
- 이모지 제거, 여러 줄바꿈 정리
- 단어 필터링, source→target 치환 규칙
- 결과 즉시 복사

### Prompt Template

자주 쓰는 프롬프트를 저장해두고 변수만 갈아끼우는 도구.

- 본문에 `{{변수}}` 삽입 (한글 변수명도 가능)
- 변수마다 치환값을 채우는 source·target 테이블
- 치환된 최종 본문을 바로 복사
- 템플릿과 치환값을 함께 저장

## 글 네비게이션

posts·update 상세글 하단에 이전글·다음글 링크를 좌우로 배치하고, 제목 위에 목록으로 돌아가는 링크를 붙였다.
